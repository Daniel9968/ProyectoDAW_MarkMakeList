# Instrucciones mínimas de reproducción
1. Primero crea la base de datos MySQL llamándola "proyectodaw" y luego importa el fichero baseDatos/proyectodaw.sql para que se cree la estructura e inserte el contenido.
2. Luego coloca esta carpeta "HTDOCS" en la raíz del servidor.
3. Ve en local a la página principal.
